# KautismSFOEditor
This world best SFO editor ever! Support UTF-8.

![DEMO](http://4.bp.blogspot.com/-1H63l4wiUAM/UPbRGxPMh9I/AAAAAAAADUA/GgPXTtwt1tA/s1600/%E6%9C%AA%E5%91%BD%E5%90%8D-1.png)

- 中文介面
- 支援UTF-8 ，支援日文、中文等遊戲標籤
- 提供一般編輯器常有的遙控遊玩編輯
- 提供解析度更改
- 提供音效設定
